    RetroFlix_Angular

    Project Overview

    RetroFlix_Angular is a single-page Angular application built with TypeScript and Angular Material. It connects to a custom-built Movie API that provides data on classic retro films. Users can register, manage their profiles, explore a catalog of movies, and create a personalized list of favorite films. The application offers detailed views of movie genres and directors. You can visit the webpage here: https://rob-cost.github.io/CF-RetroFlix_Client_Angular/welcome

    Features

    • ✅ Built with Angular CLI and TypeScript
    • ✅ Styled with Angular Material components and custom SCSS
    • ✅ User registration and authentication
    • ✅ Update and delete user profiles
    • ✅ View a catalog of retro movies with:
      • Descriptions
      • Genres
      • Director details
    • ✅ Add or remove movies from a Favorites list
    • ✅ Open genre descriptions and director bios in modal dialogs

    Usage

    Prerequisites

    • Node.js
    • Angular CLI (npm install -g @angular/cli)
    • A running instance of the Movie API (local or deployed)

    Getting Started

    1. Clone this repo:
    git clone https://github.com/rob-cost/CF-RetroFlix_Client_Angular.git
    cd CF-RetroFlix_Client_Angular
    1. Install dependencies
    npm install
    1. Start the development server
    ng serve
